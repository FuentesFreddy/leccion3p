# Consumo de API REST - JSONPlaceholder con TypeScript

Este repositorio contiene una solución limpia y modular para consumir e interactuar con servicios RESTful utilizando la API simulada de **JSONPlaceholder** sobre el recurso `/posts`. La aplicación opera e imprime de manera síncrona tanto en consola nativa como mediante una interfaz web interactiva.

## 🚀 Instrucciones de Ejecución

### Requisitos Previos
Tener instalado [Node.js](https://nodejs.org/) y el compilador oficial de TypeScript de forma global.

```bash
npm install -g typescript
